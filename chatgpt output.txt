Absolutely. You can add a **"Current Balance"** box that shows how much money you currently have based on:

**Current Balance = Total Income − Total Expenses**

### 1. Add this summary box

Inside the `.summary` section, add:

```html
<div class="summary-box" style="background:#6a1b9a;">
    <h3>Current Balance</h3>
    <h2 id="balanceTotal">$0</h2>
</div>
```

For example:

```html
<div class="card summary">

    <div class="summary-box income">
        <h3>Income</h3>
        <h2 id="incomeTotal">$0</h2>
    </div>

    <div class="summary-box expense">
        <h3>Expenses</h3>
        <h2 id="expenseTotal">$0</h2>
    </div>

    <div class="summary-box net">
        <h3>Net Savings</h3>
        <h2 id="netTotal">$0</h2>
    </div>

    <div class="summary-box" style="background:#6a1b9a;">
        <h3>Current Balance</h3>
        <h2 id="balanceTotal">$0</h2>
    </div>

</div>
```

---

### 2. Replace your `render()` calculations

Find:

```javascript
let income = 0;
let expenses = 0;
```

and change it to:

```javascript
let income = 0;
let expenses = 0;

let totalIncome = 0;
let totalExpenses = 0;
```

Then inside the `transactions.forEach(...)` loop, add:

```javascript
if(t.type === "income"){
    totalIncome += t.amount;
}else{
    totalExpenses += t.amount;
}
```

---

### 3. Update the balance display

Near the bottom of `render()`, after:

```javascript
document.getElementById("netTotal")
    .textContent =
    "$" + (income-expenses).toFixed(2);
```

add:

```javascript
const balance = totalIncome - totalExpenses;

document.getElementById("balanceTotal")
    .textContent =
    "$" + balance.toFixed(2);
```

---

### Optional: Start with your real bank amount

If you want the tracker to show your actual money rather than starting from $0, add:

```javascript
let startingBalance = 5000;
```

Then change:

```javascript
const balance = totalIncome - totalExpenses;
```

to:

```javascript
const balance =
    startingBalance +
    totalIncome -
    totalExpenses;
```

Now the website will show:

* 💰 Income this month
* 💸 Expenses this month
* 📈 Net savings this month
* 🏦 Current Balance (how much money you have total)

which is usually the most useful number to see at a glance.
Yes. If you're using the Chart.js version, you can add a second dataset that stays at the savings goal amount across all months.

Replace the `datasets` section with this:

```javascript
datasets: [
{
    label: "Balance",
    data: values,
    tension: 0.3
},
{
    label: "Savings Goal",
    data: labels.map(() => savingsGoal),
    borderDash: [10, 5],
    pointRadius: 0,
    tension: 0
}
]
```

Or, in the full chart:

```javascript
balanceChart = new Chart(ctx, {
    type: "line",
    data: {
        labels: labels,
        datasets: [
        {
            label: "Balance",
            data: values,
            tension: 0.3
        },
        {
            label: "Savings Goal",
            data: labels.map(() => savingsGoal),
            borderDash: [10, 5],
            pointRadius: 0,
            tension: 0
        }
        ]
    },
    options: {
        responsive: true,
        maintainAspectRatio: false
    }
});
```

This creates:

```text
$2000 ─ ─ ─ ─ ─ ─ 🎯 Goal Line
       \
$1500   \
          \
$1000      \____ Balance
              \
$500
```

### Make the goal line only appear when a goal exists

Instead of hard-coding the datasets, you can build them dynamically:

```javascript
const datasets = [
{
    label: "Balance",
    data: values,
    tension: 0.3
}
];

if(savingsGoal > 0){
    datasets.push({
        label: "Savings Goal",
        data: labels.map(() => savingsGoal),
        borderDash: [10, 5],
        pointRadius: 0,
        tension: 0
    });
}
```

Then:

```javascript
data: {
    labels,
    datasets
}
```

### Add a label when the goal is reached

You can also highlight points that are above the goal:

```javascript
pointRadius: values.map(
    value => value >= savingsGoal ? 8 : 4
)
```

This makes it very easy to see where your balance crosses the goal line.

A nice next step would be adding a **projected goal date** that estimates when you'll hit your savings goal based on your average monthly savings rate. That pairs very well with the goal line.
